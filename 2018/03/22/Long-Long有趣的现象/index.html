2. 总结

其实，通过源码分析：

在128~127这些数的对象在内存中被缓存起来，在内存中提供重用，这些数获取的时候就会自动获取缓存中的数，而不是new出来的对象。
所以，这与自动打包拆包无关。

再所以…，判读Long对象是否相等的时候建议不要采用 == 的形式，程序中无法绝对的判断参数值是否会在这个区间内，如果不在，那就出现让你摸不着头脑的问题。
